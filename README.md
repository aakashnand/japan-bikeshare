# Japan Bikeshare

A data pipeline that archives GBFS snapshots from Japan's public bike-sharing systems.

## Data Sources

- [ODPT GBFS Feeds](https://ckan.odpt.org/dataset/?res_format=GBFS&tags=%E3%83%90%E3%82%A4%E3%82%AF%E3%82%B7%E3%82%A7%E3%82%A2-bikeshare) — source data from Japan's Open Data Platform for Transport
- [GBFS Specification](https://github.com/MobilityData/gbfs/blob/master/gbfs.md) — General Bikeshare Feed Specification reference
